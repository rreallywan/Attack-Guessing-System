# Attack-Guessing-System
Rule Based Expert System for Cyber-Attack Identification and Mitigation System  
CYBER ATTACK IDENTIFICATION AND MITIGATION SYSTEM

PROJECT OVERVIEW This system identifies cyber attacks using user
responses and provides corresponding mitigation strategies.

FEATURES - Diagnostic questioning interface - Attack identification
logic (rule-based) - Detailed mitigation recommendations - Learning
capability via dynamic knowledge base updates - Easy-to-use Tkinter
GUI - Persistent storage using JSON

FILES INCLUDED - cyber_attack_guesser_gui.py → Main GUI application -
cyber_kb.json → Knowledge base file - README.txt → Instructions and
documentation

SYSTEM REQUIREMENTS - Python 3.8+ - Tkinter (bundled with standard
Python) - JSON support

HOW TO RUN 1. Ensure all project files are in the same folder. 2. Open
terminal or command prompt. 3. Run: python cyber_attack_guesser_gui.py
